Как настроить интеграцию:
1. Нажмите «Тест соединения» — должен вернуться профиль пользователя
2. Нажмите «Загрузить стадии» — выберите нужные стадии из списка
3. Укажите код поля «Подразделение» из вашего смарт-процесса
4. Добавьте роботы в Битрикс24 (инструкция ниже)

📖 Инструкция: настройка роботов в Б24
1. Зайдите в CRM → Смарт-процессы → Ваш процесс → Автоматизация (Роботы)
2. На стадии «Одобренные» добавьте робота «Исходящий вебхук»:
  • Адрес: скопируйте URL выше
  • Метод: POST
  • Параметры: element_id={ID}&stage_id={STAGE_ID}&secret=ВАШ_СЕКРЕТ
3. Повторите для стадий «Исполненные» и «Отклонённые»
4. Создайте новую стадию «Внеплановые расходы» (если её ещё нет)
